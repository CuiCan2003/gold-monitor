🔄 数据流程
================================================================================

用户浏览器
    │
    ├─→ GET /                       → public/index.html
    ├─→ GET /styles.css             → public/styles.css
    ├─→ GET /script.js              → public/script.js
    │
    ├─→ GET /api/prices             → api/prices.py
    │       │                          │
    │       │                          ├─→ WebSocket连接融通金
    │       │                          │   wss://rtjwbqt.jzj9999.com:8443/gateway
    │       │                          │
    │       │                          ├─→ 发送订阅帧(Base64编码)
    │       │                          ├─→ 接收二进制数据帧
    │       │                          ├─→ 解析品种和价格
    │       │                          │
    │       │                          └─→ 返回JSON:
    │       │                              {
    │       │                                "success": true,
    │       │                                "data": [
    │       │                                  {"code": "AU9999_BUY", "price": 971.0},
    │       │                                  {"code": "SILVER_BUY", "price": 11.76}
    │       │                                ]
    │       │                              }
    │       │
    │       └─→ 前端接收JSON → 更新页面显示
    │
    └─→ GET https://api.binance.com/api/v3/ticker/24hr
            │
            └─→ 获取BTC/ETH价格 → 更新页面显示


🎯 核心功能
================================================================================

✅ 实时价格获取
   - 黄金: 971元/克 (融通金WebSocket)
   - 白银: 11.76元/克 (融通金WebSocket)
   - BTC: 币安API
   - ETH: 币安API

✅ 自动单位换算
   - 克 ↔ 盎司 (1盎司 = 31.1034768克)

✅ 货币转换
   - CNY ↔ USD
   - 实时汇率API

✅ 每秒更新
   - setInterval(fetchAllPrices, 1000)

✅ 无CORS限制
   - Python后端代理
   - 部署后所有人都能访问
