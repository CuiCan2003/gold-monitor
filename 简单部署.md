# 🚀 超简单部署 - 直接用你的爬虫！

## 现在只需要3步！

### 第1步: 安装Flask

```bash
pip install flask flask-cors
```

### 第2步: 运行

```bash
cd D:\API\大宗商品价格监控
python app.py
```

### 第3步: 打开浏览器

访问: http://localhost:5000

✅ **完成！**

## 📊 API接口

### 获取所有价格
```
GET http://localhost:5000/api/prices
```

返回:
```json
{
  "success": true,
  "data": [
    {
      "time": "2025-10-20 16:00:00",

      "unit": "CNY/g"

    {
```

### 获取单个价格
```
GET http://localhost:5000/api/price/AU9999
```
